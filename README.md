## Features

### News Automation
- Hourly bilingual news posts (Arabic + English)
- AI-generated summaries via GPT-4o-mini
- Auto-created discussion threads with tags
- Multi-source: Enab Baladi, Koora

### Debate System
- Karma voting with upvote/downvote reactions
- AI auto-tagging for topic detection
- Hot tag based on activity levels
- Monthly and all-time leaderboards

### Moderation
- Ban/allow users from debates
- Thread close/reopen controls
- Case logging and appeal system
- Open discussion thread (no karma)

---

## Commands

| Command | Description |
|---------|-------------|
| `/karma [user]` | View karma stats |
| `/disallow <user>` | Ban user from debates |
| `/allow <user>` | Unban user from debates |
| `/close [reason]` | Close debate thread |
| `/open` | Reopen closed thread |
| `/rename <name>` | Rename debate thread |
| `/cases [search]` | Search moderation cases |
| `/toggle` | Enable/disable posting |
